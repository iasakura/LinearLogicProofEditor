# Linear logic proof editor

demo: https://iasakura.github.io/LinearLogicProofEditor/

## usage

| Formula | Text notation |
|---|---|
| !A  | !A  |
| ?A  | ?A  |
| A⊥ | ~A  |
|(A ⊗ B)| A\*B|
|(A & B)|	A&B|
|(A ⊕ B)|	A\|B|
|(A ⅋ B)| A@B|

- Click variable: apply axiom
- Click operator: apply right rule
- Click comma between formula: apply cut rule
